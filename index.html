✅ Use Zoho CRM

• Agent records — consortia, region, destination focus, status
• Activity logging — calls, emails, follow-ups, events
• Pipeline for booking conversion tracking
• Automated follow-up reminders (48hr webinar, 5-day roadshow)
• Reports by rep, consortia, region, destination
• Mobile app for reps on the road
• Email integration — send survey, log replies automatically
• Kanban for lapsed advisor reactivation pipeline
